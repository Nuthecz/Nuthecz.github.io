Android Studio配置

Gradle下载配置

​ 直接更换国内腾讯镜像源，打开gradle - wrapper -gradle-weapper.properties 进行更改。然后点击 Sync Now 进行同步。参考-> Android导入项目时Gradle下载速度慢\超时\失败解决方法_导入gradle项目特别慢-CSDN博客

​ 但是这样也是很慢，还是得等，有时候还会突然跑到源地址去下载，搞不明白。(后续补充：有时候改了国内源，然后停止加载zip文件，之后重试一下就快很多了)

生成APK

  • build -> Generate Signed Bundle or APK：选择APK，然后创建key(注意需要路径完整)，之后再选择release即可

存储空间管理

  • 移动 .gradle 到指定位置
    • 将 .gradle文件 从C盘移动到D盘 - tc310 - 博客园 (cnblogs.com)，这里同时还需要相应修改 idea.plugins.path 和 idea.log.path
  • 移动 .android 到指定位置
    • https://blog.csdn.net/weixin_42768634/article/details/115142632

报错汇总

  • 遇到问题 Cannot use connection to Gradle distribution 'https://mirrors.cloud.tencent.com/gradle/gradle-8.2-bin.zip' as it has been stopped.

    • 这里直接关闭项目，再重新打开即可。

  • Error running 'app': Default Activity not found

    • 这里修改 configuration，更改 Launch Options -> Launch:nothing
    • android studio怎么运行没有activity的service、broadcastReceiver、cotentProvicer等_androidstudio 如何启动服务?-CSDN博客

  • 日志不能在 As 中显示

    • 检查算法助手有没有 hook 对应程序，如果存在，那么它好像开机自启，自动将日志捕获了。关闭应用 hook 就可以显示日志了
