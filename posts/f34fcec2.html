  3. add the path command which is export PATH=~/...../bin:$PATH to ~/.bashrc

How to use gdb

  • path of gdb /home/tcz/environment/android-ndk-r21e/prebuilt/android-arm64/gdbserver/gdbserver -> /data/local/tmp/gdbserver64
  • 这里需要注意gdbserver的架构需要与调试的程序架构相同，所以在~/.bashrc中设置环境变量alias gdb-arm='~/environment/android-ndk-r21e/prebuilt/linux-x86_64/bin/gdb'
  • 同时根据android-ndk-r21e的gdb只有python2.7的特性，需要配置特定python2的gef-legacy，这里就需要在~/.gdbinit中加入set auto-load safe-path ~/android/arm/arm_study允许特定目录加载自己的.gdbinit文件，然后在那个文件中填写source /home/tcz/pwn/tools/gef-legacy/gef.py进行gdb调用，这样就可以在这个目录下实现gdb-arm调用gef了

程序调用

​ 下面是程序调用的语句，参数少于4个使用r0~r3传参，参数大于4个时，多的参数使用栈传递，同时使用r0传递返回值，lr传递返回地址

汇编指令

  • 寻址方式

    • 立即数寻址 -> mov r0, #2
    • 寄存器寻址 -> mov r0, r1
    • 寄存器移位寻址
      • LSL 逻辑左移 -> mov r0, r1, LSL #4 r0 = r1 << 44
      • LSR 逻辑右移
      • ROR 循环右移
      • ASR 算术右移
      • RRX 扩展的循环右移
    • Load Store 的寻址方式
      • ldr(保存到寄存器) 与 str(保存到内存)
      • 立即数
      • 寄存器 -> str r0,[r3] *(r3) = r0
      • 基址变址 -> ldr r0,[r3, #4] r0 = *(r3+4)
    • 栈寻址
      • stmfd
        • push {r5, r6, r7, lr}
        • stmfd sp!, {r5-57, lr} 和上面等价
      • ldmfd

  • 汇编指令

    • 跳转指令
      • B 强制跳转指令
      • BL 带返回的跳转指令，lr寄存器存储返回地址
      • BLX 带返回和带状态切换的跳转指令，arm -> thumb，thumb->arm
      • BX 带状态切换的跳转指令
    • 数据处理指令
      • mov, add, sub, and, eor, orr, bic
      • mov r0，r1
      • add rO，r1, r2 -> r0 = r1 + r2
      • sub r0，r1, r2
      • and rO，r1, r2 -> r0 = r1 & r2
      • eor r0，r1，r2 -> r0 = r1 ^ r2
      • orr rO，r1, r2 -> r0 = rr2
      • bic r0，r1，#0xf -> 0x12345678 -> 0x12345670
    • 乘法指令
      • MUL rO，r1, r2 -> r0 = r1 x r2
      • MLA rO，r1, r2, r3 -> r0 = r1 x r2 + r3
      • SMULL r0，r1，r2，r3 -> r0 = (r2 x r3)的低32位，r1 = (r2 xr3)的高32位
      • SMLAL r0，r1，r2，r3 -> r0 = (r2 x r3)的低32位 + r0，r1 = (r2 x r3)的高32位 + r0
      • UMULL r0，r1，r2，r3 -> r0 = (r2 x r3)的低32位，r1 = (r2 xr3)的高32位 -> 无符号
      • UMLAL r0，r1，r2，r3 -> r0 = (r2 x r3)的低32位 + r0，r1 = (r2 x r3)的高32位 + r0 -> 无符号

  • 内存访问指令

    • ldr 4字节读取
    • ldrb 1字节读取
    • ldrh 2字节读取
    • str 4字节写入
    • strb 1字节写入
    • strh 2字节写入
